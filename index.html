How To Create Personal GitHub Page

1. First of all go to github.com and create an account.
2. After creating an account click on the top right hand corner to get into your repository.
3. In the repository section you'll want to create a new reposity by clicking the green "New" sign.
4. Type in username.github.io and create a repository.
5. Once you have created a repository you'll need to code the website with Visual Studio.
6. After coding a website to your liking save the files.
7. Go to your repository page on GitHub and copy the files you created into your GitHub repository and commit.
8. After you commit you can view your website by typing in your username.github.io in the browser.
9. If everything was done right you should see your website.
